== Description ==

Dengan plugin ini anda bisa menampilkan Rate / Kurs Rupiah Indonesia terhadap mata uang asing.

Data yang ditampilkan merupakan tampilan realtime dari website masing-masing bank tersebut.

Beberapa bank yang telah dimasukkan adalah:

   1. BCA
   2. Mandiri
   3. BNI
   4. Panin

Tentu saja ini akan memudahkan anda untuk membandingkan kurs masing-masing bank tersebut ;)

Feature yang ada pada versi awal ini adalah:

    * Menampilkan wp-kurs dimana saja pada theme anda.
    * Menampilkan wp-kurs pada post anda.
    * Menampilkan wp-kurs dengan Wp Kurs Widget (beta).

== Installation ==

Anda hanya butuh ini agar di aktifkan dari dashboard anda :)

Cara pemakaian:

   1. Pemakaian pada theme.
      Silahkan pastekan code berikut di dalam theme anda:
      <? wp_kurs(); ?>
   2. Pemakaian pada post.
      Untuk pemakaian pada post, anda cukup menuliskan kode berikut:
      [wpkurs]
   3. Pemakaian widget.
      Untuk widget anda tinggal konfigurasikan pada widget ;)
